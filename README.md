Este proyecto consiste en el desarrollo de un agregador de noticias utilizando el framework Scrapy. Se han implementado tres spiders independientes para extraer información de distintos medios digitales gallegos: El Ideal Gallego, La Opinión A Coruña y La Voz de Galicia. Cada spider se adapta a la estructura HTML específica de su página web, utilizando selectores CSS y XPath según el caso. De cada noticia se extraen campos comunes como el título, la fecha, un resumen y la fuente, aplicando una limpieza básica de los datos para eliminar espacios innecesarios y evitar valores nulos. El resultado es un conjunto de datos unificado y estructurado, exportable en formato JSON.
